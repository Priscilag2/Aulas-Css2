*/ !important O !important é uma regra do CSS utilizada para adicionar importância a uma propriedade/valor do CSS. Esta regra substituirá todos os estilos que já foram aplicados no elemento referente a propriedade que a regra está sendo adicionada. só utilize a !important se for necessario. Pode ser usado em qualquer lugar no seu projeto e afeta todos os valores de propriedades css anteriores. Exemplo: css css color:red !important; /* A cor da fonte vai ser vermelha */ color:blue; /* O texto acima terá a cor azul pois o !important sobrescreveu a cor vermelha*/ html*/
